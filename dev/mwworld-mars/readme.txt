Current world used for mars world 

Place this world in the world folder for the on the server folder of Minecraft tools downloaded below

It can then be started from the Jave version of minecraft 

The server and python development API can be downloaded from here
https://sourceforge.net/projects/python-with-minecraft-windows/

It’s important to make sure the spigot server for the minecraft server is upto date here is a useful repository
https://yivesmirror.com/downloads/spigot


The mars world was originally obtained from here
https://www.planetminecraft.com/project/welcome-to-mars---a-map-based-on-the-martian/
